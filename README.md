这个项目主要面向OA流程页面审批交互页面的基本建模、报表打印。并非用于vue可视化构建工具
### 项目启动
```
npm install
npm run serve
```

[后台代码](https://github.com/medlar01/edit-project.git)

### 功能
* 页面编辑
* 打印模式与编辑模式切换
* 后台构建vue组件
* 实现sql与api方式交互数据
* TODO oa系统对接
